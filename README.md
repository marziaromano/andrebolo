# Andrea Bolognino portfolio site

Static site ready for GitHub Pages.

## Files
- `index.html` - main page
- `style.css` - responsive layout and visual style
- `script.js` - gallery and fullscreen lightbox
- `assets/images/` - images rendered from the portfolio PDF
- `portfolio.pdf` - downloadable original PDF
- `CNAME` - custom domain for GitHub Pages

## Deploy on GitHub Pages
Upload/replace these files in the repository root, then check:
Repository -> Settings -> Pages -> Deploy from branch -> main / root.
